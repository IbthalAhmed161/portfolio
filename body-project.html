Body Performance Classification System

A machine learning project for classifying physical performance levels and predicting broad jump distance using body measurements and fitness indicators.

13,393Original rows
13,386Clean rows

Problem

Physical performance data contains patterns that can help classify fitness levels and estimate performance outcomes. The goal was to build reliable classification and regression models.

My Approach

  • Verified data quality and removed invalid records.
  • Handled outliers using IQR capping.
  • Created engineered features such as BMI and strength index.
  • Compared multiple ML models for classification.
  • Used regression to predict broad jump distance.
Results

Model outcomes

Best Classifier

Random Forest achieved the best classification performance.

Best Regression

Linear Regression performed best for broad jump prediction.

Strong Features

Flexibility, sit-ups, strength index, body fat, age, and BMI were important.

Reliable Evaluation

Cross-validation showed stable model performance.
