Analytics + ML Project

Hotel Booking Data Analysis & Cancellation Prediction

An end-to-end analytics project built around a real business question: why bookings get cancelled and how cancellation risk can be predicted. It combines Python EDA, SQL exploration, Power BI dashboarding, and XGBoost modeling.

Worked with 86,639 cleaned records, identified a 27.7% cancellation rate, and built an XGBoost model reaching 81.8% accuracy with ROC-AUC around 0.88.
